Installing Notepadlet

You can install Notepadlet by dragging the following link to your bookmarks toolbar:

Notepadlet - Untitled

This creates a bookmarklet with an empty Notepadlet that you can use any time, anywhere. Even on offline.

Note that this page is a fully-functional Notepadlet, so you can just start editing this site or bookmark it directly.

Notepadlet saves itself automatically every time you click outside the paper.
